# CompTIA Security & Network Hands-on Labs 🛡️

This repository contains documentation and scripts from hands-on laboratory exercises based on the CompTIA A+ (Core 1 & Core 2) certification standards. It demonstrates practical experience in network configuration, system administration, endpoint security, and IT service management (ITSM).

## 📂 Repository Structure

### 1. [01-Networking-and-Protocols](./01-Networking-and-Protocols)
Hands-on configurations of fundamental networking concepts.
* **Key Topics:** Manual IP/Subnet configurations, VPN (SSTP) setup with EAP-MSCHAP v2, and network troubleshooting using `traceroute`.

### 2. [02-Windows-System-Administration](./02-Windows-System-Administration)
Managing Windows environments and enforcing access controls.
* **Key Topics:** Active Directory Organizational Units (OU), user account lifecycle management, and implementing the Principle of Least Privilege (PoLP) via NTFS permissions.

### 3. [03-Endpoint-and-Group-Policy](./03-Endpoint-and-Group-Policy)
Centralized security management and system hardening.
* **Key Topics:** Implementing Group Policy Objects (GPO), configuring local firewall profiles (Domain/Private/Public), and enforcing strict password policies.

### 4. [04-Cybersecurity-and-Defender](./04-Cybersecurity-and-Defender)
Defending against malware and analyzing system vulnerabilities.
* **Key Topics:** Microsoft Defender configuration (Exclusions & Scans), identifying Social Engineering exploits (Phishing, Whaling), and analyzing workstation security settings (UAC, BitLocker, Rogue Accounts).

### 5. [05-Linux-Administration](./05-Linux-Administration)
Core Linux command-line operations for system administrators.
* **Key Topics:** User privilege management (`su -`, locking accounts), file directory operations, and modifying ownership (`chown`, `chgrp`).

### 6. [06-Scripting-and-Automation](./06-Scripting-and-Automation)
Writing scripts to automate routine IT tasks and monitor resources.
* **Key Topics:** 
  * **PowerShell:** Execution policies, error handling (Try...Catch), and control structures (If/For) for VM automation.
  * **Bash:** Utilizing command substitution, piping (`grep`, `awk`, `cut`), and output redirection to monitor disk space.

### 7. [07-ITSM-and-Incident-Response](./07-ITSM-and-Incident-Response)
Simulating enterprise Help Desk workflows and compliance standards.
* **Key Topics:** Managing IT support tickets, handling Personally Identifiable Information (PII) securely, and responding to unauthorized software installations based on standard operating procedures (SOP).
